
**Update `readConfig`:** read all three fields, default missing → `null`. **`writeConfig`** unchanged structurally.

**Update `validateRepoPath` (rename to `validateLocalRepo`):**
- If path does NOT exist yet → return `{ ok: true }` (will be created by clone)
- If path is non-empty AND not a git repo → fail (don't risk overwriting user data)
- If path is a git repo → ok (clone skipped, pull will run)

**New function `validateRepoUrl(url: string): ValidationResult`:**
- Empty → fail "Repo URL is required"
- Doesn't match `/^(https?:\/\/|git@)[\w\-.@:/~]+/i` → fail "Invalid URL format"
- Else ok

**New function `validateRulesTarget(p: string): ValidationResult`:**
- Empty → fail
- Not absolute → fail
- Else ok (created on demand)

## Task 3 — Replace `runUpdateHandler` with `runSyncHandler`

**Files:** `src/main/ipc.ts`, `tests/main/ipc.test.ts`

Remove `runUpdateHandler`, `RunUpdateDeps`, `platformMatches` (no longer needed — Sync uses current OS automatically).

**New `runSyncHandler`:**
```ts
export type RunSyncDeps = {
  currentPlatform: NodeJS.Platform
  configPath: string
  emit: (line: LogLine) => void
}

export async function runSyncHandler(deps: RunSyncDeps): Promise<RunResult> {
  const cfg = readConfig(deps.configPath)
  if (!cfg.repoUrl) return fail('Repo URL not configured')
  if (!cfg.repoPath) return fail('Local repo path not configured')
  if (!cfg.rulesTarget) return fail('Rules target folder not configured')

  const u = validateRepoUrl(cfg.repoUrl); if (!u.ok) return fail(u.error)
  const p = validateLocalRepo(cfg.repoPath); if (!p.ok) return fail(p.error)
  const t = validateRulesTarget(cfg.rulesTarget); if (!t.ok) return fail(t.error)

  return withRunLock(async () => {
    // Step 1: clone or pull
    const isExisting = existsSync(join(cfg.repoPath!, '.git'))
    if (!isExisting) {
      deps.emit({ time: now(), text: `$ git clone ${cfg.repoUrl} ${cfg.repoPath}`, level: 'info' })
      // ensure parent dir exists
      mkdirSync(dirname(cfg.repoPath!), { recursive: true })
      const clone = await runCommand('git', ['clone', cfg.repoUrl!, cfg.repoPath!], {
        cwd: dirname(cfg.repoPath!),
        onLine: deps.emit,
      })
      if (clone.exitCode !== 0) return failWithExit('git clone failed', clone.exitCode, deps)
    } else {
      deps.emit({ time: now(), text: '$ git pull', level: 'info' })
      const pull = await runCommand('git', ['pull'], { cwd: cfg.repoPath!, onLine: deps.emit })
      if (pull.exitCode !== 0) return failWithExit('git pull failed', pull.exitCode, deps)
    }

    // Step 2: detect and run install script
    const isWin = deps.currentPlatform === 'win32'
    const scriptName = isWin ? 'install.ps1' : 'install.sh'
    const scriptPath = join(cfg.repoPath!, scriptName)
    if (!existsSync(scriptPath)) {
      return fail(`${scriptName} not found in repo root`)
    }
    deps.emit({ time: now(), text: `$ ${scriptName} (RULES_TARGET=${cfg.rulesTarget})`, level: 'info' })
    const env = { RULES_TARGET: cfg.rulesTarget! }
    const inst = isWin
      ? await runCommand('powershell', ['-ExecutionPolicy', 'Bypass', '-File', scriptPath], {
          cwd: cfg.repoPath!, env, onLine: deps.emit,
        })
      : await runCommand('bash', [scriptPath], { cwd: cfg.repoPath!, env, onLine: deps.emit })
    if (inst.exitCode !== 0) return failWithExit('install failed', inst.exitCode, deps)
    deps.emit({ time: now(), text: '✓ DONE (exit 0)', level: 'success' })
    return { ok: true, exitCode: 0 }
  }).catch((e: Error) => ({ ok: false, exitCode: -1, error: e.message }))
}

function fail(error: string): RunResult { return { ok: false, exitCode: -1, error } }
function failWithExit(prefix: string, exitCode: number, deps: RunSyncDeps): RunResult {
  deps.emit({ time: now(), text: `✗ ${prefix} (exit ${exitCode})`, level: 'error' })
  return { ok: false, exitCode }
}
```

**Update `registerIpc`:** rename `run-update` channel to `run-sync`. Update `set-config` to validate all three fields. Keep `get-config`/`pick-repo-path`/`get-platform` unchanged.

Tests: rewrite ipc tests for `runSyncHandler` covering: missing URL/path/target, fresh clone, existing pull, install missing, env passed correctly.
